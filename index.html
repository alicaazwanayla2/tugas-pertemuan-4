Summary Git

Apa Itu Git?

Git Adalah Sistem Version Control yang Digunakan Untuk Melacak Perubahan Pada File Proyek Git Membantu Programmer Bekerja Secara Kolaboratif Tanpa Perlu Kehilangan Data Versi Sebelumnya

Cara Kerja Git

Git Bekerja Dengan Tiga Area Utama:

  • Working Directory: tempat file diedit.
  • Staging Area: tempat menandai file yang siap disimpan.
  • Repository: tempat menyimpan seluruh riwayat perubahan (commit).

Perintah Dasar Git

git init -- membuat repository baru
git status -- menampilkan status dari file
git add -- menambahkan semua file ke staging area
git commit -m "message"-- menyimpan perubahan ke repository lokal
git log -- melihat riwayat commit
git branch -- melihat atau membuat branch baru\
git checkout <branch> -- pindahke branch
git merge <branch> -- menggabungkan branch
git push origin main -- mengunggah perubahan ke GitHub
git pull origin main -- mengambil update dari GitHub

Langkah Praktik

  1. Buka terminal di folder proyek dan jalankan git init
  2. Tambahkan file dengan git add .
  3. Simpan perubahan dengan git commit -m "pesan commit"
  4. Hubungkan dengan GitHub: git remote add origin <url>
  5. Kirim ke GitHub dengan git push origin main
  6. Gunakan git pull untuk sinkronisasi dengan tim.
